# 模块

模块可以定义函数，类和变量，也可能包含可执行的代码

# 模块的作用

1. 当函数，类和变量很多时，可以很好的进行管理
2. 把同一类型的功能代码，写入到一个模块文件中，既方便管理，也方便调用
3. 一个模块就是一个工具包

# 包

模块越来越多，放在一起不好管理，包是模块集合
包的结构示意图
from 包名 import 模块名
import 包名。模块名
from 包名。模块名 import 函数 | 类 | 变量

包可以有多个层级
