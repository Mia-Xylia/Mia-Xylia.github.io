# 成员方法

类中定义的函数也称为成员方法 / 方法

1. 方法定义的参数列表中有 self
2.self 时定义成员方法时，需要写上的
3.self 表示当前对象本身
4. 通过对象调用方法时，self 会隐式的传入
5. 在方法内部，需要用 self 才能访问到成员变量

python 支持动态的增加方法

# self

在成员方法内，访问对象的属性 / 成员变量
不写 self，需要使用 @staticmethod 变为静态方法

哪个对象调用，self 指的就是哪个对象

# 作用域

局部变量：成员方法中定义的变量，作用域在它的方法中
属性的作用域为整个类
属性和局部变量可以重名，有 self 表示属性，没有表示局部变量
