# 函数

# 入门

为完成某一功能的程序指令的集合
函数分为系统函数（内置函数，模块中提供的函数）和自定义函数

  • 好处
    1. 提供的代码的复用性
    2. 将实现的细节封装起来，供其他函数调用
  • 基本语法
    基本语法
    调用方式：函数名（实参列表）

# 函数的注意事项和使用细节

1. 函数的参数列表可以是 0 - 多个
2. 函数要遵循命名规则
3. 函数的变量是局部的
4. 函数名相同时，调用最近的函数，就近原则
5. 传参方式是位置传参，数据类型不用指定
6. 可以有多个返回值，返回值的类型不被限定
7. 支持关键字参数（形参名 = 实参值）
8. 支持默认参数，指定参数时以指定为主，默认参数放在参数列表后
9. 支持可变参数，传入的 0 - 多个实参会被组成一个元组 def sum (*args)
10. 传入的多个关键字参数时会被组成一个字典 def sum (**arg)
11. 调用另外一个文件的函数 import 文件名 文件名。函数名 ()

# lambda 匿名函数

如果函数作为参数使用，但只使用一次，我们可以考虑 lambda 匿名函数

  • 基本语法
    1.lambda 形参列表：函数体（一行代码）
    2.lambda 关键字，表示定义匿名函数

# 全局变量和局部变量

全局变量：在整个程序范围内都可以访问，定义在函数外，拥有全局作用域的变量
局部变量：只能在其被声明的函数范围内使用，定义在函数内部，拥有局部作用域的变量

  • 注意事项
    1. 未在函数内部重新定义 n1，使用全局变量 n1
    2. 在函数内部重新定义 n1，就近原则，使用函数内部重新定义的 n1
    3. 在函数内部使用 global 关键字标明使用全局变量
