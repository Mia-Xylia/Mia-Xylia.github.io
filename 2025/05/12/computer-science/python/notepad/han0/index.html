# python 语言的概述

# 程序的概念

计算机执行某些操作或者解决某个问题而编写的一系列有序指令的集合

# 特点

简单易学，面向对象，开源，可移植性，解释性语言，高级语言，可扩展性，丰富的库，规范的代码

# python 快速入门

1.python 程序后缀是.py
2.print 表示输出 ("") 中的内容
3.# 表示注释内容

# Python 运行原理

运行原理

# python 程序开发注意事项

1.python 源文件通常以.py 为后缀，但不必须
2. 默认情况顺序执行
3.python 区分大小写
4.python 语句不需要；结束，规范是不写

# python 常用快捷键

shift + delete 删除当前行
ctrl + d 复制当前行
ctrl + / 注释选中行
ctrl + alt + L 快速代码格式化
ctrl + h 查看层级关系
ctrl + r 运行
ctrl + f 查找

# 注释 comment

1. 单行注释用#
2. 多行注释用 """/’’’
3. 在开头指定编码格式 #coding: 编码（新版没有用）

# 规范和文档

1.tab 键向右移动，shift+tab 整体向右移动
2.= 两边用空格
3. 变量用，隔开

python官方文档

https://docs.python.org/zh-cn/3.12/
