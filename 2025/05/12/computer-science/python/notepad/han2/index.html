# 标志符的命名规则和规范

凡是可以自己取名字的地方都叫标识符

  • 规则
    1.26 个英文字母大小写，0-9，_组成
    2. 数字不可以开头
    3. 不可以使用关键字，但能包含关键字
    4. 不可以包含空格
    5. 区分大小写
    int 不是关键字
  • 规范
    1. 变量要小写，多个单词用下划线分开，常量全部大写
    2. 函数名要小写，多个单词用下划线分开，私有函数以下划线开头
    3. 类名使用大驼峰命名
