# 数据类型

基本数据类型：int float bool string
a，b 是变量，没有类型

# 整数类型

  • 细节
    1.int 可以表示很大的数
    2. 十六进制 0x 开头，八进制 0o 开头，二进制 0b 开头

一个字节占八位
int 占字节是一个变量，每一次增量是四个字节
可以通过 sys.getsizeof 查看变量的大小

# 浮点类型

# 表现形式

1. 科学计数法 5.12e2
2. 十进制数的形式 0.512 .512
浮点数计算后存在精度损失，可以使用 decimal 类进行精确计算

# 布尔类型

  • 基本使用
  • 细节
    1. 布尔类型只有两个值：true 和 false
    2. 布尔类型可以和其他数据类型进行比较，比如数字，字符串，在比较时，python 会将 true = 1，false = 0
    3. 在 python 中，非 0 被视为真值，0 被视为假值

# 字符串类型

  • 注意
    1.python 不支持单字符输出
    2. 用三个单引号 / 双引号是字符串保持原样使出
    3. 在字符串前加‘r’可以使整个字符串不会被转义

# 字符串驻留机制

python 仅保留一份相同且不可变字符串，不同的值被存放在字符串的驻留池当中，python 的驻留机制对相同的字符串只保留一份拷贝，后续创建相同字符串时，不会开辟新空间，而是把该字符串的地址赋给新常见的变量

  • 发生驻留机制条件
    1. 由大小写字母，0-9，_组成
    2. 长度为 0/1
    3. 字符串在编译时进行驻留，而非运行时
    4.[-5 ,256] 的整数数字
    但是 pytharm 对驻留机制进行了改进，提升效率，节约内存
