# 使用对比

  • python 文件
    python 文件运行的话，块是所有代码，每次运行都是从头开始运行
    优点：通用，传播方便，适用于大型项目
    缺点：需要从头运行
  • pycharm 控制台
    python 控制台以每一行（任意行）为一块（Shift+Enter）
    优点：可以显示每个变量属性
    缺点：不利于代码阅读及修改
  • jupyter
    Jupyter 以任意行为块运行的
    优点：利于代码阅读及修改
    缺点：环境据要配置
