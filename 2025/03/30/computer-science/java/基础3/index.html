# 赋值运算符

将某个运算后的值赋给指定的变量
= += -= *= /= %=

  • 细节
    ・运算顺序：从右向左
    ・赋值运算符左边只能是变量，右边可以是变量，表达式，常量值
    ・复合赋值运算符会进行类型转换（强制类型转换）

# 标识符

标识符：对各种变量，方法和类等明明是使用的字符序列
自己可以取名
命名规则：
组成：26 个英文字母大小写，0-9，_,$
不能以数字开头
不能使用关键字和保留字
区分大小写
不能包含空格

  • 练习
  • 规范
    1. 包名：多单词组成时所有字母都小写 aaa.bbb.ccc
    2. 类名，接口名：多单词组成，每个单词首字母大写
    3. 变量名，方法名：第一个单词首字母小写，其他大写
    4. 常量名：所有字母大写

# 二进制

二进制（0b/oB 开头），十进制，八进制（0 开头），十六进制（0x/0X 开头）

  • 二进制转十进制
    从最低位开始，每位数 * 2^(位数 - 1), 求和
  • 八进制转十进制
    从最低位开始，每位数 * 8^(位数 - 1), 求和
  • 十六进制转十进制
    从最低位开始，每位数 * 16^(位数 - 1), 求和
  • 十进制转二进制
    不断除以 2，直到商为 0，把余数倒过来
  • 十进制转八进制
    不断除以 8，直到商为 0，把余数倒过来
  • 十进制转十六进制
    不断除以 16，直到商为 0，把余数倒过来
  • 二进制转八进制
    从低位开始，每三位一组，转成相应的八进制数
  • 二进制转八进制
    从低位开始，每四位一组，转成相应的八进制数
  • 八进制转二进制
    每一位转成一个三位的二进制数
  • 十六进制转二进制
    每一位转成一个四位的二进制数

# 位运算符

# 原码，反码，补码

对于有符号的而言：
1. 二进制的最高位是符号位，0 表示正数，1 表示负数
2. 正数的原码，反码，补码都一样
3. 负数的反码 = 它的原码符号位不变，其他位取反
4. 负数的补码 = 反码 + 1 反码 = 补码 - 1
5.0 的反码，补码都是 0
6.java 没有无符号数，换言之，Java 中的数都是由符号的
7. 在计算机运算的时候，都是以补码的方式来运算的
8. 当我们看运算结果的时候，要看它的原码

# 位运算详解

  • 第一组
    按位与 & 两位全为 1，结果为 1，否则为 0
    按位或 | 两位有一个为 1，结果为 1，否则为 0
    按位异或 ^ 两位一个 0 一个 1，结果为 1，否则为 0
    按位取反～0->1 1->0
  • 第二组
    1.>>：算术右移，低位溢出，符号位不变，并用符号位补溢出的高位
    2.<<：算数左移，符号位不变，低位补 0
    3.>>>：逻辑右移即无符号右移，低位溢出，高位补 0
