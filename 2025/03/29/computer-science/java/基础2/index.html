# 变量介绍

变量是程序的基本组成单位
变量的三个基本要素：类型 + 名称 + 值

  • 变量概念
    变量相当于内存中一个数据存储空间的表示
    1）声明变量
    2）赋值
    3）使用
  • 变量入门
  • 变量细节
    1. 变量表示在内存中的一个存储区域 [不同的变量，类型不同，占用空间大小不同]
    2. 该区域有自己的变量名和数据类型
    3. 变量必须先声明再使用
    4. 该区域的数据 / 值可以在同一类型范围内变化
    5. 变量在同一个作用域内不能重名
    6. 变量 = 变量名 + 值 + 数据类型

强类型语言的表现

    浮点类型：float（4）,double（8）

# 整数类型

  • 存储空间
    byte -128~127
    short -（215）~(215)-1
    int -（231）~（231）-1
    long -（263）~（263）-1
  • 细节
    1. 整型有固定的范围和字段长度，不受 os 影响
    2. 整型常量默认为 int 型，声明 long 型常量后加‘l’或‘L’

3. 变量常声明为 int 型，除非不足以表示大数，才使用 long
4.bit：计算机中最小存储单位
byte：计算机中基本存储单元
1byte = 8bit

# 浮点数

float -3.4.3E38~3.403E38
double -1.798E3.8~1.798E308
浮点数 = 符号位 + 指数位 + 位数位
尾数部分可能丢失，造成精度损失（小数都是近似值）

  • 细节
    1. 有固定的范围和字段长度，不受 os 影响
    2. 常量默认为 double 型，声明 float 型常量后加‘f’或‘F’

3. 浮点型表示方式：
十进制形式： 5.12 5.12f 0.512 = .512
科学计数法： 5.12e2（5.12×10^2） 5.12E-2 (5.12÷10^2)
4. 通常情况下应该使用 double 型，因为它更精确
5. 浮点数使用陷阱：2.7 和 8.1/3 比较

# 字符型

char 单个字符
string 多个字符
字符类型可以存放一个数字

  • 细节
    1. 字符常量用‘’
    2. 允许使用转义字符
    3.char 的本质是一个整数，在输出时对应的是 unicode 码对应的字符
    4.char 类型可以参与运算
  • 本质
    1. 字符型存储到计算机中，将字符对应的码值找出来，转化为二进制进行存储
    2. 字符和码值的对应关系是通过字符编码表决定的

# 布尔型

没有 null，适用于逻辑运算
不能用 0/1 代替 false/true

# 编码

ASCII 一个字节表示，128 个字符
Unicode 使用两个字节表示字符，字母和汉字都是占两个字节
utf-8 字母用一个字节表示，汉字用三个字节表示
gbk 字母一个字节，汉字两个字节
gb2312 表示汉字
big5 码 繁体汉字

# 自动类型转换

精度：char - int - long - float - double
容量：byte - short - int - long - float - double

  • 注意和细节
    1. 多种类型混合运算时，自动转换成容量最大的进行计算
    2.（byte，short）和 char 不能自动转换
    3.byte，short，char 参与计算时首先转换成 int 类型
    4.boolean 不参与转换

# 强制类型转换

自动类型转换的逆过程，使用时加上强制转换符（），可能造成精度降低或者溢出

  • 细节
    1. 强转符号只针对最近的操作数有效
    2.char 可以保存 int 的常量，但不能保存 int 的变量
    3.byte 和 short 类型要先变成 int 类型
