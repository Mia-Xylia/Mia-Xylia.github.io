  • java 流程分析
    .java 源文件
    .class 字节文件

  • 开发细节
    1.java 源文件是以.java 为扩展名的。源文件的基本组成部分是.class 文件
    2.java 区分大小写
    3. 以；结尾
    4.main 方法有固定书写格式：public static void main (String [] args)
    5. 一个源文件最多只能有一个 public 类，其他类的个数不限制，每一个类对应一个.class 文件
    6. 大括号成对出现
    7.public 类的类名 = 文件名
    8. 一个源文件中最多只有一个 public 类。其他类的个数不限，也可以将 main 方法写在非 public 类中，指定运行非 public 类，入口方法就是非 public 类的 main 方法

  • 路径
    相对路径：从当前目录开始定位，形成的路径
    绝对路径：从根目录 / 顶级目录开始定位，形成的路径

  • 常用的 doc 命令
