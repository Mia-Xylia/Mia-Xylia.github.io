    Android 工程师

  • java 故事
    1995 年第一版
    8**，11 LTS
    java se 标准版，java EE 企业版，java me 小型版

  • java 特性
    1.oop
    2. 健壮的（强类型机制，异常处理，垃圾的自动收集等）
    3. 跨平台 .java --> .class --> Windows/Linux/Mac
    4. 解释型语言：编译后不能直接运行
    编译型语言：c/c++

  • 编译工具的介绍
    editplus
    notepad++
    Subline Text
    IDEA
    eclipse

  • java-jdk 介绍
    JVM（java 虚拟机）实现了跨平台，包含在 jdk 中

JDK = JRE + java 的开发工具 [java,javac,javadoc,javap 等]
JRE = JVM + java 核心类库
