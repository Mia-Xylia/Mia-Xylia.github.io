# 数组

数组可以存放多个同一类型的数据。数组是一种数据类型，是引用类型。
1. 数组中可以存放任何数据类型
2. 数组创建后，如果没有赋值，有默认值 int/short/byte/long 为 0
float/double 为 0.0，char \u0000,boolean false,String null
3. 数组始于引用类型，数组型数据是对象
