# if else

让分支有选择的运行

# 单分支

if（条件表达式）{
执行代码块
}

# 双分支

if（条件表达式）{
执行代码块
}
else{
执行代码块 2
}

# 多分支

if（条件表达式）{
执行代码块 1
}
else if (条件表达式){
执行代码块 2
}
…………
else{
执行代码块 n
}
多分支可以没有 else，如果所有的条件表达式都不成立则一个执行入口都没有
如果有 else，如果所有的条件表达式都不成立，则默认执行 else

# 嵌套分支

# switch

switch (表达式){
case 常量 1:
语句块 1;
break;
case 常量 2:
语句块 2;
break;
………………
case 常量 n:
语句块 n;
break;
default:
default 语句块:
break;
}
如果没有 break 语句，执行下一个语句块，不判断是否匹配

  • 细节
    1. 表达式数据类型应该和 case 后常量类型一致，或者可以自动转成可以比较的类型
    2.switch 中表达式的返回值必须是 （byte，short，int，char，enum，String）
    3.case 中的值必须是常量或常量表达式，不能是变量
    4.default 是可选的，没有匹配的 case，执行 default
    5.break 语句用来在执行完一个 case 分支后使程序跳出 switch 语句块，如果没有写 break，程序会顺利运行到 switch 结尾，除非执行到 break
  • 课堂练习

判断具体的数值不多且符合 byte，short，int，char，enum，String，建议使用 switch
区间判断 /boolean 使用 if else

# for

for (循环变量初始化；循环条件；循环变量迭代){
循环操作（可以多条语句）;
}

  • 细节
    1. 循环条件是返回一个布尔值的表达式
    2.for (; 循环哦按段条件；) 中的初始化和变量迭代可以写到其他地方，但是两边的；不能省略
    3. 循环初始化可以有多条初始化语句，但要求类型一样，并且中间用，隔开
    循环变量迭代也可以有多条变量迭代语句，中间用，隔开
