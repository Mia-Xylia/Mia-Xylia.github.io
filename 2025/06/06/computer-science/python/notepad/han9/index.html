# 封装

封装就是把抽象的数据（属性）和对数据的操作（方法）封装在一起，数据被保护在内部
程序只有通过被授权的操作，才能对数据进行访问

# 封装的好处

1. 隐藏实现细节
2. 可以对数据进行验证，保证安全合理
3. 保护数据隐私，要求授权才可以访问

# 私有成员

公共的类和方法在类的内部和类的外部都可以正常使用

# 属性 / 方法私有化

类中的方法或变量 +__开头命名，只能在内部使用

# 访问私有属性 / 方法

提供公共的方法，用于对私有成员的操作

# 继承

解决代码复用的问题，更靠近人类思维
多个类存在相同的属性 / 方法时，可以从这些类中抽象出父类，在父类中定义这些相同的属性和方法，所有子类不需要重新定义这些属性和方法
示意图

# 注意事项和细节

1. 子类继承了所有的属性和方法，非私有的属性和方法可以在子类直接访问，但是私有属性和方法不能在子类直接访问，要通过父类提供公共的方法去访问
2. 在 python 中，object 是所有其他类的基类 ctrl+H
3. 支持多继承
5. 在多继承中，基类有同名成员，左边比右边优先级高

# 调用父类成员

子类和父类出现同名的成员 父类名称 /super ()

# 基本语法

2.super (). 成员变量 super (). 成员方法 (self)

模块 abc 为定义抽象类提供了基础，不需要实例化

# 注意事项和细节

1. 抽象类需要继承 ABC，并且需要至少一个抽象方法
2. 抽象类中可以有普通方法
3. 如果一个类继承了抽象类，则必须实现抽象类的所有抽象方法，否则还是抽象类
